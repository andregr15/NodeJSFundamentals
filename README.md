**NodeJS Fundamentals**

*Project to learn the fundamentals of NodeJS*
1. To run the project do a clone or download the zip file, after that access the folder on terminal
or cmd and type:

```bash
yarn

or

npm install
```

2. After that type:

```bash
yarn dev:server

or

npm run dev:server
```

3. To run the tests run

```bash
yarn test

or

npm run test
```

4. To execute some request use Insomnia or Postman
